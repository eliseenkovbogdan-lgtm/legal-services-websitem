Решение алгоритмической задачи

Условие задачи

Задача: Определить, можно ли из заданных длин сторон a, b, c построить треугольник, и если да, то определить его тип (остроугольный, прямоугольный, тупоугольный).

Математическая модель:

Треугольник существует, если сумма любых двух сторон больше третьей:

  • a + b > c
  • a + c > b
  • b + c > a

Тип треугольника определяется по теореме косинусов:

  • Прямоугольный: a² + b² = c² (или другие комбинации)
  • Остроугольный: a² + b² > c² (для всех сторон)
  • Тупоугольный: a² + b² < c² (для одной из сторон)
