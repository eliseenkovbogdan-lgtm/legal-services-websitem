Инструкция по созданию скриншота:

  1. Нажмите кнопку "Получить данные из DOM"
  2. Откройте Инструменты разработчика (F12)
  3. Перейдите в раздел Console
  4. Введите: document.getElementsByTagName('p')[2].childNodes[1].innerText
  5. Сделайте скриншот и сохраните его
  6. Добавьте скриншот на эту страницу

Здесь будет размещен ваш скриншот с результатом выполнения команды в Console
